# BitCam 1.1 - Multi-Version & 1.21.4 Support

This release introduces expanded multi-version compatibility, specifically adding full support for newer Minecraft versions (up to 1.21.4) along with several API adaptations for the latest modding toolchains.

## ✨ New Features & Compatibility
* **Extended Minecraft Support:** Added full support for the latest Minecraft versions (including 1.21.4).
* **Multi-Version Preprocessing:** Implemented extensive preprocessor macros to maintain a single codebase across multiple Minecraft versions without breaking legacy support.

## 🛠️ Technical Changes
* **Rendering Compat:** Updated client render compatibility to gracefully handle changes in newer Minecraft versions.
* **Keybindings & Commands (Fabric & NeoForge):** Updated `KeyMapping` categories, command registrations, and entity feature renderer callbacks to be compatible with both legacy and the latest API changes.
* **Network Payloads (Fabric):** Adapted `PayloadTypeRegistry` registration for newer Fabric API versions (`serverboundPlay` and `clientboundPlay`).
* **Settings Screen:** Minor fixes and cleanups in the settings screen logic.
